# NeyroIMG
Нейросеть, которая обучается на основе нарисованных вами чисел.

При запуске начните рисовать цифры. После предсказания нейросети укажите правильно ли она ответила ( нажмите на одну из кнопок: кнопка правильно и кнопка не правильно) и укажите цифру, которую вы написали. После 5 повторении написания определенной цифры нажмите на кнопку Обучить нейросеть. Проверьте нейросеть)
